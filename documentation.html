API Documentation

This is the documentation for an API which will store information on movies and allow users to sign up and favorite movies of their choosing.

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get a list of all movies to the user /movies GET None A JSON object holding data about all of the movies.
Get data about a single movie by title /movies/[title] GET None A JSON object holding data about a single movie containing a title, genre, director, and release year.

{ title: 'Alien', director: 'Ridley Scott', year: '1979', genre: 'Science-Fiction' }
Get a list of all genres to the user /genres GET None A JSON object holding data about all of the genres.
Get data about a movie genre /genres/[name] GET None A JSON object holding data about a single genre containing a name and description:

{ name: 'Horror', description: 'Films that seeks to elicit fear for entertainment purposes' }
Get a list of all directors to the user /directors GET None A JSON object holding data about all of the directors.
Get data about a director /directors/[name] GET None A JSON object holding data about a single genre containing a name and D.O.B.:

{ name: 'Ridley Scott', born: '1937' }
Add new user accounts /users POST A JSON object holding data about the user to add:

{ Username: "User", Password: "password", Email: "user@gmail.com" }

 A JSON object holding data about the user that was added, including an ID:

{ id: 1, Username: "User", Password: "password", Email: "user@gmail.com" }
Update a user's information /users/[username] PUT A JSON object holding data about the user to add:

{ Username: "User", Password: "password", Email: "user@gmail.com" }

 A text message indicating the user's information was updated.
Add movies to user's list of favorites /users/[username]/favorites/[title] POST None A message indicating the movie was successfully added to the user's list of favorites.
Remove movies from user's list of favorites /users/[username]/favorites/[title] DELETE None A message indicating the movie was successfully removed to the user's list of favorites.
Remove user accounts /users/[ID] DELETE None A message indicating the user's account was succesfully deleted.
